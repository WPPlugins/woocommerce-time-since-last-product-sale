Plugin extends WordPress WooCommerce by adding time since last sale to product page.

== Description ==

* Plugin extends WordPress WooCommerce by adding time since last sale to product page.
* Customizable text, style and position on product page.
* Option to display only to admin.
* Option to select number of days to cover when calculating time since last sale. Option to select order status to cover.
* If no sales occuried during covered period - no text is displayed.
* Ready to use widget.
* Option to enable/disable on all product pages by default.
* [TIMELASTSALE] shortcode.
* <?php echo get_last_sale_time_lite(); ?> function.
* All options are in "Admin panel -> WooCommerce -> Time Since Last Sale"

== Installation ==

1. Upload the entire 'woocommerce-time-since-last-sale' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Time since last sale info will be automatically added to every product page.
